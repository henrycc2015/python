##### 应用场景
> 相邻模块间7px的间距

##### 用法
```
<template>

    <v-stick>本案需您补充理赔申请材料原件后支付赔款</v-stick>
    <stub></stub>
    <v-stick :closable='false'>本案需您补充理赔申请材料原件后支付赔款</v-stick>
    
</template>

```

> 该组件是透明背景色
